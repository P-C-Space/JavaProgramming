= Arrays, ArrayLists, and Records

== Array Details
- 배열은 [] 표현, 배열의 범위를 초과한다면 ArrayIndexOutOfBoundsException
- Java의 배열은 객체이므로 배열 변수는 배열만 참조 가능, 배열을 포함하지는 않는다.

=== For-each Loops
- 배열의 모든 요소를 반환
- 값을 반환하기 때문에 수정을 바람직하지 않다
[source, java]
----
int[] intList = new int[10];
for ( int item : intList ){
    item = 17; // 잘못된 방법
}
----
